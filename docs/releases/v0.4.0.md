# v0.4.0

**Release Date:** 2026-02-14

Feature release adding subtitle generation from STT transcription results and extensible configuration maps for provider-specific settings.

## Highlights

- Subtitle generation from STT transcription results
- Extensible config maps for provider-specific settings

## Added

- Subtitle package for SRT/VTT generation from transcription results ([`17730a7`](https://github.com/agentplexus/omnivoice/commit/17730a7))
- Configurable max characters per line and lines per cue for subtitles ([`17730a7`](https://github.com/agentplexus/omnivoice/commit/17730a7))
- Word-level timestamp-based cue splitting ([`17730a7`](https://github.com/agentplexus/omnivoice/commit/17730a7))
- Extensions map in TranscriptionConfig for provider-specific STT settings ([`84c37f5`](https://github.com/agentplexus/omnivoice/commit/84c37f5))
- Extensions map in SynthesisConfig for provider-specific TTS settings ([`665c3be`](https://github.com/agentplexus/omnivoice/commit/665c3be))

## Fixed

- Subtitle `wrapText` no longer clips words when text exceeds line limit ([`63144bb`](https://github.com/agentplexus/omnivoice/commit/63144bb))

## Tests

- Provider conformance testing utilities for call system providers ([`9683ca2`](https://github.com/agentplexus/omnivoice/commit/9683ca2))
- Provider conformance testing utilities for transport providers ([`9683ca2`](https://github.com/agentplexus/omnivoice/commit/9683ca2))

## Documentation
