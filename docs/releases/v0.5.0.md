# Release Notes: v0.5.0

**Release Date:** 2026-02-28

## Highlights

- **Organization Rename**: GitHub organization renamed from `agentplexus` to `plexusone`

## Breaking Changes

| Component | Before | After |
|-----------|--------|-------|
| Go module | `github.com/agentplexus/omnivoice` | `github.com/plexusone/omnivoice` |

## Migration Guide

Update your import paths:

```go
// Before
import "github.com/agentplexus/omnivoice/tts"

// After
import "github.com/plexusone/omnivoice/tts"
```

Update your `go.mod`:

```bash
go mod edit -droprequire github.com/agentplexus/omnivoice
go get github.com/plexusone/omnivoice@v0.5.0
go mod tidy
```

## Installation

```bash
go get github.com/plexusone/omnivoice@v0.5.0
```
